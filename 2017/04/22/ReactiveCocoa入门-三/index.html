# 映射

map：对源信号发出的数据做修改，传入一个block，类型是返回对象，参数是value，不用包装成信号，直接返回修改后的数据。 flattenMap：用于信号中信号，当源信号发出数据时，返回一个新的信号，但是订阅者收到的是返回信号中的值，信号可以是任意类型。

### concat (链接)

-> 按一定顺序拼接信号，当多个信号有可能发送数据的时候，按顺序进行接收。（每个都接收）
-> 借用个形象的比喻，可以理解成各个信号间是皇上和太子关系，只关注当前皇上。当上一个信号（当前皇上）sendCompleted（驾崩）了则后一个信号（当前太子）变成新的皇上，从此以后不需要关注之前的皇上了。
